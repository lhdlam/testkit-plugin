# testkit

AI test-automation toolkit (Claude Code + Cursor) for testers — web (Playwright) & desktop (PySide6).

The plugin lives in [`plugins/testkit/`](plugins/testkit/). See its [README](plugins/testkit/README.md)
and [USAGE](plugins/testkit/USAGE.md).

## Install (Claude Code)

```
/plugin marketplace add lhdlam/testkit-plugin
/plugin install testkit@lhdlam
```
